# 🖼️ HennessyNight 可视化启动指南

## 📍 第一步：打开 Docker Desktop

### 选项 1：从开始菜单打开

```
1. 按键盘上的 Windows 键（或点击屏幕左下角的 Windows 图标）
   ┌─────────────────────────────────┐
   │  🔍 搜索框                      │
   │  输入: Docker Desktop           │
   └─────────────────────────────────┘

2. 点击搜索结果中的 "Docker Desktop"
   ┌─────────────────────────────────┐
   │  🐋 Docker Desktop              │
   │  应用                           │
   └─────────────────────────────────┘

3. Docker Desktop 窗口会打开
```

### 选项 2：从桌面或任务栏打开

```
桌面图标:
┌──────────┐
│   🐋     │
│  Docker  │
└──────────┘
双击打开

任务栏图标:
═════════════════════════════════════
... 📁 🌐 🐋 ...
    点击 Docker 鲸鱼图标
═════════════════════════════════════
```

### ✅ 确认 Docker 已启动

在 Docker Desktop 窗口中，左下角会显示：

```
成功状态:
┌─────────────────────────────────────────┐
│ Docker Desktop                          │
│                                         │
│ ● Docker Desktop is running            │  ← 绿色圆点
│                                         │
│ Containers (0)                          │
│ Images (0)                              │
└─────────────────────────────────────────┘
```

**或者**任务栏的 Docker 图标：
- ❌ **启动中**: 鲸鱼图标有动画效果
- ✅ **已就绪**: 鲸鱼图标静止不动

⏱️ **请等待 30-60 秒让 Docker 完全启动**

## 📂 第二步：运行启动脚本

### 方法 A：双击运行（推荐，最简单）

#### 步骤 1：打开文件资源管理器

```
按键盘快捷键:  Win + E
或者点击任务栏的文件夹图标: 📁
```

#### 步骤 2：导航到项目文件夹

在地址栏输入或粘贴：
```
D:\Hennessy.Night\HennessyNight
```

然后按 **回车键**

#### 步骤 3：找到启动脚本

在文件列表中找到这个文件：
```
文件名: quick-start.bat
图标:   📄 (带齿轮的窗口图标)
类型:   Windows 批处理文件
大小:   约 2 KB
```

#### 步骤 4：双击运行

**双击** `quick-start.bat` 文件

会弹出黑色的命令提示符窗口：

```

│ C:\Windows\System32\cmd.exe                      │

│ ✅ Docker 正在运行                               │
```

## ❌ 如果遇到问题

### 问题 1：Docker 相关错误

```
❌ Docker 未运行
📝 请先启动 Docker Desktop，然后重新运行此脚本
```

**解决办法**：
1. 确认已打开 Docker Desktop
2. 等待 Docker 完全启动（左下角绿色 "running"）
3. 重新双击 `quick-start.bat`

---

### 问题 2：端口被占用

```
Error: Port 3000 is already in use
```

**解决办法**：
1. 关闭其他占用 3000 端口的程序
2. 或者修改 `.env` 文件中的 `PORT=3000` 为其他端口（如 3001）

---

### 问题 3：浏览器显示"无法访问此网站"

**检查清单**：
- [ ] 命令行窗口是否还在运行？（不要关闭它）
- [ ] 是否看到 "正在启动开发服务器" 消息？
- [ ] 等待 30 秒后再刷新浏览器
- [ ] 确认地址是 `http://localhost:3000`（不是 https）

---

## 🛑 如何停止服务器

在命令行窗口中：

1. 按 `Ctrl + C` 键
2. 如果询问 "终止批处理作业吗？"，输入 `Y` 然后回车
3. 窗口会关闭，服务器停止

---

## 📞 需要帮助？

如果上述步骤遇到问题：

1. 截图错误信息
2. 查看 `START.md` 文件中的故障排查章节
3. 检查 Docker Desktop 是否正常运行
4. 确认网络连接正常（需要下载依赖）
